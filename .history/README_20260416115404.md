## 🎯 CallStudent Pro | 智能课堂加权点名系统


让课堂提问告别“全凭运气”，进入“算法公平”时代。

这是一款专为教师设计的沉浸式课堂互动工具。它不仅解决了传统点名随机性过强的问题，还通过加权概率算法、多学科惩罚库以及沉浸式动效，极大地提升了课堂活跃度。
------------------------------
## ✨ 核心亮点## 1. ⚖️ 真正的加权随机算法
不同于普通的 Math.random()。支持通过 Excel 导入学生权重。

* 低权重：适用于基础薄弱或性格内向的学生，保护其受挫心理。
* 高权重：增加特定学生的被选概率，确保课堂互动的针对性。

## 2. 🎭 沉浸式互动体验

* 视觉震慑：仿老虎机式的滚动特效，配合随机生成的背景浮动姓名，营造紧张刺激的课堂氛围。
* 听觉反馈：基于 Web Audio API 实时合成 Tick 倒计时音效与 Boom 中奖音效，内置语音合成（TTS）自动播报中奖名单。

## 3. 🧪 学科定制化惩罚库
内置语、数、外、理、化、生、政、史、地九大核心学科惩罚内容。

* 多层惩罚机制：抽中惩罚后支持“再抽一次”，并设有“命中注定”拦截逻辑，增加趣味性。
* 动态编辑：支持最简单格式的文本解析，教师可一键批量修改全学科惩罚配置。

## 4. 📂 极致的文件管理

* 多班级管理：支持批量导入 Excel，自动识别班级名称，一键流畅切换。
* 历史闭环：右侧实时记录已提问同学，支持手动撤销。数据持久化存储于本地（LocalStorage），刷新不丢失。

------------------------------
## 🏗️ 架构设计 (重构后)
项目采用领域驱动设计 (DDD) 思想进行组件拆分，确保了极高的可扩展性：

src/
├── hooks/        # 【大脑】封装点名算法(useRoll)、音频(useAudio)、持久化(useStorage)
├── components/   # 【皮肤】Atomic Design 风格，含 Business(业务) 与 Modals(弹窗)
├── utils/        # 【插件】Excel 解析器、加权随机算法工具
├── constants/    # 【内存】存储学科配置与内置惩罚文案
└── types.ts      # 【契约】严格的 TypeScript 类型约束

------------------------------
## 🚀 快速开始## 导入要求
Excel 文件名即为班级名。表格格式如下：

| 姓名 (第一列) | 权重 (第二列) |
|---|---|
| 王小明 | 1 |
| 李华 | 3 |

## 键盘快捷键 (即将到来)

* Space: 开始/停止点名
* Enter: 抽取惩罚
* Esc: 关闭所有弹窗

## 🛠️ 技术实现细节

* 状态管理：自定义 useStorage Hook 结合 useEffect 实现状态与本地存储的无损同步。
* UI 性能：通过 Portal 思想管理 Modals，并使用绝对定位确保核心按钮在任何屏幕尺寸下保持垂直水平居中。
* 类型安全：启用 verbatimModuleSyntax，确保模块导入的纯净性。
